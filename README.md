## 開発環境

* Java 1７
* Android Studio
* server 環境
  * [DAIgoAPI](https://github.com/bvlion/DAIgoAPI) を実行する

## 配信環境

* master にマージすると GitHub Actions で APK が App Distribution に配信される
* v タグを切ることで本番デプロイが行われる
